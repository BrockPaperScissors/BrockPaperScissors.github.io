Team Deets

A Match history tool made for League of Legends players. Users can enter their Riot ID and tag, and the app will utilize the Riot API's to display several statistics of each member in the users last 5 matches.

  • Created using ReactJS, utlizing React hooks, React Router, and third-party API's.
  • Used external data from API's to create meaningful metrics that help players analyze their performance.
